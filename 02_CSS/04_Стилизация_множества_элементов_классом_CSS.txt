Стилизация множества элементов с помощью класса CSS

Классы позоволяют вам использовать один и тот же CSS стиль для множества HTML-элементов. Вы можете видеть это применив класс red-text к первому элементу. 

Тесты: 

1. Ваш первый h2-элемент должен быть красный
2. Ваш h2-элемент должен иметь класс red-text 
3. Ваш первый p-элемент должен быть красный 
4. Ваш второй и третий p-элементы не должны быть красными
5. Ваш первый p-элемент должен иметь класс red-text
