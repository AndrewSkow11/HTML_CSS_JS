Установите семейство шрифтов для элемента

Вы можете установить какой шрифт для элементов следует использовать, используя семейство font-family. 

Для примера, если вы хотите установить для h2-элемента шрифт sans-serif (без засечек), вам следует использовать следующий CSS: 

h2 {
    font-family: sans-serif;
}

Сделайте все ваши p-элементы моноширным шрифтом (monospace)

Тесты: 

Ваш p-элемент должен иметь шрифт monospace.
