Изменение размера шрифта элемента

Размер шрифта контролируется CSS-свойством font-size, выглядящем так: 

h1 {
    font-size: 30px; 
}

В том же самом теге <style> который содержит ваш red-text класс, создайте вход для p-элемента и установите font-size в 16 пикселей (16px).

Тесты:

Между тегами style, дайте p-элементу font-size 16px. Браузер и увеличение текста должны быть в 100%.
